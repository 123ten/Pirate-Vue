`Pirate Admin` 是一个 通用的后台管理模板，由 `Vue3` + `Ts` + `Vite4` + `Tailwindcss` + `VSCode`(可选 `webstorm`) 实现。它使用了最新的前端技术栈，自适应PC端和移动端。

`Pirate Admin` 内置了 `约定式路由`、`layout布局`、`请求封装`、`请求拦截`、`登录拦截`、`Tailwindcss`、`i18n多语言` 等基础功能。
